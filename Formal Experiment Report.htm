2 Introduction

The web site defacto.at belongs to APA De Facto, a daughter company of APA, the Austria Press Agency. The De Facto web site is an interface to an online archive of more than 70 million documents from over 190 sources (mostly newspaper and newswire articles).

The APA De Facto web site is aimed at media researchers wishing to stay abreast of developments and trends in the media or to research background information on a particular topic.

APA De Facto would like to simulate new customers registering for the web site and performing a few typical tasks. A series of formative and summative tests with novice users were planned to help APA De Facto redesign their site to be more intuitive to first time users. Since the web site is in German, the studies were carried out in German with German-speaking test users.

3.1 Test Procedure

This test, the initial benchmark test, BM1, gathered concrete performance and rating data from the initial version of the web site. It was run as a formal experiment of a single interface (summative evaluation). More details on the methods of usability testing can be found in Keith Andrews' lecture notes [Andrews, 2011]. The structure of this report broadly follows the Common Industry Format (CIF) [ISO, 2006].

{Description of the test data. What data was measured or gathered?}

One pilot user and 16 real test users were recruited. Users were given a fictitious identity to register for the web site. Users then performed a series of typical tasks. Performance data and successful completion data was collected for each task. At the end of the test, each user was interviewed to gather feedback. Finally, a feedback questionnaire was administered to gather subjective rating data from each user.

The data was analysed to determine mean and standard deviation for the benchmark BM1 test. The results from BM2 (after a web site redesign) will later be compared with the results from BM1.

3.2 Test Users

Friends and colleagues were recruited for each study, with the following characteristics:

  • Experienced users of computers.
  • Experienced users of the web.
  • Native speakers of German (or excellent command of German if non-native).
  • Age 18 or over.

Summary of the important background infos. Sixteen university students volunteered for this study (12 men and 6 women). They ranged in age from 18 to 36 (median 22) years. The average web use for the participants was 8.4 (median 5) hours per week, 83% used the web a few times per week or more.

3.6 Interview Questions

The following questions were asked of each user immediately following the final task.

  1. Aké to bolo?

    The user was then allowed and encouraged to talk, until they dried up and stopped of their own accord.

  2. Bolo niečo naozaj dobré, čo ste si všimli?

  3. Bolo niečo naozaj zlé, čo ste si všimli?

  4. Finally, any questions which arose during the test with that particular test user were asked.
